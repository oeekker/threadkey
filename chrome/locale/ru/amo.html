ThreadKey

Basic Information

Name ThreadKey
Summary Нажмите 'T' для группировки по обсуждениям, 'U' для группировки не по обсуждениям, 'G' для группировки по методу сортировки.

Add-on Details

Description

Очень простое расширение, позволяет включать группировку по обсуждениям нажатием клавиши 'T', группировку по методу сортировки нажатием клавиши 'G' и отменять группировку нажатием клавиши 'U'.

Даже если Вы не используете группировку по обсуждениям часто, это расширение может быть полезным если Вы случайно нажали клавишу 'G'. Вы можете переключать режимы группировки нажатием одной клавиши.

Thunderbird bug 275871 был "исправлен" так что кнопка 'G' более не включает группировку по методу сортировки. Версия 0.5 отменяет это изменение.

Учтите: обычно клавиша 'T' привязана к меню "Переход → Следующее → непрочитанное обсуждение". Для предотвращения конфликта комбинация изменена на 'Shift+T'.

Technical Details

Developer Comments

Известна ошибка с Thunderbird 2.* на Mac OS X. Изменение T/U отменяет действие T/U во всем приложении. Временную версию для Mac с комбинациями Cmd+T/Cmd+U можно скачать по ссылке "Показать всю историю версий".
