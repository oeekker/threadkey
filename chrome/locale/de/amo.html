ThreadKey

Basic Information

Name ThreadKey
Summary Drücken Sie Taste 'T' um nach Thema zu gruppieren, 'U' um nicht zu gruppieren, 'G' um nach Sortierung zu gruppieren.

Add-on Details

Description

Eine sehr einfache Erweiterung, die erlaubt „Gruppiert nach Thema“ durch Drücken von Taste ‚T‘ auszuführen, „Gruppiert nach Sortierung“ durch Drücken von Taste ‚G‘ und „Nicht gruppiert“ durch Drücken von Taste ‚U‘.

Auch wenn Sie die Gruppierung nach Thema nicht sehr oft benutzen, so kann diese Erweiterung doch nützlich sein, wenn Sie zufällig die ‚G‘ Taste gedrückt haben. Sie können dann die Gruppierungsmodus nur mit einem Tastendrück umschalten.

Thunderbird Bug 275871 war “korrigiert”, sodass ‘G’ Taste nicht mehr „Gruppiert nach Sortierung“ einschaltet. Ich habe die Version 0.5 gemacht, die dieses Verhalten wiederherstellt.

Wichtige Anmerkung: die Taste ‘T’ ist an die Menüpunkt ‚Navigation → Nächste → Ungelesenes Thema‘ gebunden: das wird auf ‚Umschalt - T‘ umgebunden

Technical Details

Developer Comments

Diese Erweiterung hat ein bekanntes Fehler unter Thunderbird 2.* auf Mac OS X. Nach umdefinieren von T/U wird T/U im ganzen Thunderbird abgeschaltet. Eine temporäre Version mit alternativen Kombinationen Cmd+T/Cmd+U für Mac ist unter “ Die vollständige Versionsgeschichte ansehen” verfügbar.
