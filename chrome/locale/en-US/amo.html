ThreadKey

Basic Information

Name ThreadKey
Summary Strike the 'T' key to sort messages by thread, strike the 'U' key to sort them unthreaded again, just like the 'G' key switches to Grouped By Sort view

Add-on Details

Description

This is a very simple extension that let the user strike the 'T' key and get the Threaded view, just like the 'G' key switches to Grouped By Sort view by default. You can also strike the 'U' key in order to get back the Unthreaded view.

Even if you don't use threaded view a lot, it can come in handy, if you happen to press 'G' by accident. This extension helps you to easily switch back to threaded or Unthreaded/Ungrouped view with a single keypress.

Thunderbird bug 275871 was "fixed" and this means that the 'G' key doesn't put Thunderbird into Grouped By Sort view anymore. I have released version 0.5, so people using my extension can continue to switch to Grouped By Sort view with the 'G' key...

Please note that the 'T' key is currently assigned to the "Go ? Next ? Unread Thread" menu item: this key assignment will be replaced by 'Shift+T'.

Technical Details

Developer Comments

There is a known error with this extension with Thunderbird 2.* on Mac OS X. Redefining T/U effectively disables T/U throughout Thunderbird. There is a temporary version with alternate shortcut keys Cmd+T/Cmd+U for the Mac available for download under See All Versions and on the ThreadKey Homepage...
