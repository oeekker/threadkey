ThreadKey

Basic Information

Name ThreadKey
Summary 按下“T”键将消息按主题分组，按下“U”键取消按主题分组，就像“G”键切换按排序分组视图一样

Add-on Details

Description

这是一个非常简单的扩展，它允许用户通过按下“T”键获得按主题分组的视图，就像默认情况下“G”键切换到按排序分组的视图一样。你也可以通过按下“U”键取消按主题分组的视图。

即使你不经常使用按主题分组的视图，这个扩展也会在你不小心按下“G”键时起到作用。这个扩展帮助你仅点击一次按键，就能方便地切换到按主题分组或者取消按主题分组/取消分组的视图。

Thunderbird bug 275871 已被“修正”，这意味着“G”键不会再切换 Thunderbird 到按排序分组视图。我发布了 0.5 版本，这样使用此扩展的用户可以继续使用“G”键切换到按排序分组的视图...

注意“T”键现在被分配给了“前往下一个未读主题”菜单项：这个按键将被替换为“Shift+T”。
