ThreadKey

Basic Information

Name ThreadKey
Summary Druk op de 'T' toets om berichten op conversatie te groeperen, druk op de 'U' toets om ze weer ongegroepeerd weer te geven, net als je met 'G' gegroepeerd op soort weergeeft.

Add-on Details

Description

Dit is een heel eenvoudige extensie, waardoor je als gebruiker op de 'T' kunt drukken om de berichten gegroepeerd op conversatie weer te geven, net zoals je met de 'G' toets wisselt naar gegroepeerd op soort. Je kan ook op de 'U' toets drukken om de berichten ongegroepeerd weer te geven.

Zelfs als je niet vaak op conversatie groepeert kan dit handig zijn, als je per ongeluk op de 'G' toets drukt. Deze extensie helpt je om eenvoudig terug te kunnen schakelen naar groepering op conversaties/geen conversaties met een enkele toetsaanslag.

Thunderbird bug 275871 is "fixed" en dit betekent dat je met de 'G' toets niet meer standaard gegroepeerd op soort weergeeft. Ik heb versie 0.5 gemaakt, zodat mensen die mijn extensie gebruiken de 'G' toets kunnen blijven gebruiken om naar gegroepeerd op soort te kunnen schakelen...

N.B. De 'T' toets is op dit moment toegekend aan het "Ga → Volgende → Ongelezen conversatie" menu item: deze toets zal vervangen worden door 'Shift+T'.

Technical Details

Developer Comments

Er is een bekend probleem met deze extensie bij Thunderbird 2.* op Mac OS X. Door de T/U toetsen te herdefiniëren, worden deze effectief uitgeschakeld in heel Thunderbird. Er is een tijdelijke versie met alternatieve toetsenbord snelkoppelingen Cmd+T/Cmd+U voor de Mac beschikbaar onder Volledige versiegeschiedenis bekijken en op de ThreadKey homepage...
